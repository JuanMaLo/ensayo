Definición

Es el nuevo estándar básico de HTML, lanzado en el año 2014 (según wikipedia); destacandose de los demás estandares por tener una semántica muy flexible basada en el estudio de las páginas web ya publicadas donde se encontro un estructura bastante general que explicaremos con mas detalle más adelante. Actualmente se encuenta en la version 5.2 publicada el 14 de diciembre del 2017.

Para poder entrar en materia vamos a definir que es HTML.

HTML significa Hypertext Markup Language, que traducido al español diria Lenguaje de Marcado para Hipertextos, el cual es el elemento de construcción más básico en una página web, usado para crear y representar una página web, es decir, determina y estructura el contenido en una página web.

Hypertext: Se refiere a enlaces que conectan dentro o fuera de la página web, el cual puede ser, a otra página web o site web.

Markup: Es referente al marcado para anotar texto, imágenes u otros elementos contenidos que se muestran en el navegador web.

Language: Como tal es la referencia a un lenguaje exclusivo, ya que posee su sintaxis propia, asi como una semántica muy definida.

Pero que diferencia a el html5 de las demás versiones:

Semántica:
- Permite describir con mayor precisión el contenido de la página web.
Conectividad:
- Permite comunicarse con el servidor de formas nuevas e innovadoras.
Sin conexión y almacenamiento:
- Permite a las páginas web almacenar datos localmente en el lado del cliente y operar sin conexión de manera más eficiente.
Multimedia:
- Nos otorga un excelente soporte para utilizar contenido multimedia como lo son audio y video nativamente.
Gráficos y efectos 2D/3D:
- Proporciona una amplia gama de nuevas características que se ocupan de los gráficos en la web como lo son canvas 2D, WebGL, SVG, etc.
Rendimiento e Integración:
- Proporciona una mayor optimización de la velocidad y un mejor uso del hardware.
Acceso al dispositivo:
- Proporciona APIs para el uso de varios compomentes internos de entrada y salida de nuestro dispositivo.
CSS3:
Nos ofrece una nueva gran variedad de opciones para hacer diseños más sofisticados.
