Media Queries (Consulta de medios)

Son consultas de las características del medio (dispositivo) donde se esta visualizando un documento Web y nos sirve para definir estilo definido para un mejor experencia en ese medio, esto lo hace una herramienta fundamental para implementar el "Responsive Web Design" (versión español, versión original).

Los Media Queries, a modo simple, lo que hacen es que en CSS, si están frente a un tal medio deberían hacer algo, si bien, CSS no es un lenguaje de programación, para hacer los Media Queries tenemos algunos Operadores lógicos para su implementación que son los siguientes:

Operador AND
- Las dos condiciones deben cumplirse para que se evalúe como verdadera.
Operador NOT
- Es una negación de una condición. Cuando esa condición no se cumpla se aplicarán las media queries.
Operador ONLY
- Se aplican las reglas solo en el caso que se cunpla cierta circunstancia.
Operador OR
- No existe como tal, pero puedes poner varias condiciones separadas por comas y cuando se cumpla cualquiera de ellas, se aplicarán los estilos de las media queries.
Operador IF
- Si la declaración es verdadera en el navegador web, emplea el código CSS que hay dentro de esta declaración.

Una de las recomendaciones generales para un correcto Responsive Web Design es usar la filosofía Mobile First esto significa que primero debemos pensar en los dispositivos móviles, pero lo que determina el correcto uso de los media queries, siempre sera tu usuario final, es decir, el público al que esta dirigido.
