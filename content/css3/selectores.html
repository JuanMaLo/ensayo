Id y Class

Class (Clase '.')

Una clase es una declaración que se va estar usando constantemente en una página web. Estas clases generalmente tienen un nombre despúes del punto (.rojo) con el que se identificara para poder hacer uso de ella.

Id ('#' almohadilla)

Un id se usa para afectar con un estilo único a un elemento dentro de una página web.

Especifidad

Le especifidad, es la manera en la que los navegadores deciden que valores de una propiedad CSS, son más relevantes para un elemento, para poder ser aplicados, esta basada en la regla de coicidencia que están compuestas por diferentes tipos de selectores de CSS. Si quieres aprender más sobre la especifidad, da click al siguiente enlace especifidad a profundidad.

El peso de un selector se determina de la siguiente manera:

  1. Selector "id" (1-0-0).
  2. Selector "class" (0-1-0)
  3. Selectorde tipo (0-0-1)
